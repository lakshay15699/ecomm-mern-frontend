# Frontend (React + Vite + Tailwind)

## Setup
```bash
cd frontend
npm install
npm run dev
```
> Create a `.env` with `VITE_API_URL` (your backend URL, e.g., `https://your-backend.onrender.com`).

### Tailwind is preconfigured
- `tailwind.config.js`, `postcss.config.js` and `index.css` are included.
